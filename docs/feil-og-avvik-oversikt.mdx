# Feil og avvik - oversikt

Bruk denne siden som inngang når en pakke ikke kan spores som forventet, eller utstyret ikke fungerer som det skal.

## Velg riktig spor

- [Mangler ankomstregistrering](./mangler-ankomstregistrering)  
  Pakken skal være hos dere, men mangler lokasjon eller registrering i hylle.
- [PDA-problemer](./pda-problemer)  
  Skjermen fryser, touch virker ikke eller PDA starter ikke riktig.
- [Printer-problemer](./printer-problemer)  
  Etiketter skrives ikke ut eller skriveren kobler ikke til PDA.

## Kontrollpunkter

- Start alltid med den siden som matcher symptomet best.
- Bekreft om problemet gjelder én pakke, én prosess eller alt utstyr.
- Fullfør pågående prosess i PDA hvis det er mulig før du starter feilsøking.
