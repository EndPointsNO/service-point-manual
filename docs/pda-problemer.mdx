:::

## Kontrollpunkt

- Vent til PDA-en er helt startet før du prøver på nytt.
- Start ikke samme utlevering flere ganger parallelt hvis skjermen bare bruker litt tid.

## Vanlige feil

- Skjermen virker, men skanning eller meny oppfører seg feil: lukk prosessen og start på nytt.
- Problemet gjelder bare utskrift eller tilkobling: se [Printer-problemer](./printer-problemer).
