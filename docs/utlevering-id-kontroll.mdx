# Utlevering med ID-kontroll

## Når brukes dette?

Når PDA viser `ID Kontroll` og krever kontroll av mottaker eller fullmektig før utlevering.

## Mål

Bekrefte riktig identitet før pakken signeres og utleveres.

## Steg for steg

### Steg 1: Skann eller tast inn hentekode

Start utleveringen på vanlig måte med kundens hentekode eller QR-kode.

![TODO: Skjermbilde – hentekode før ID-kontroll](/img/pda/utlevering-id-kontroll-01-skann-hentekode.png)

### Steg 2: Finn og skann riktig kolli

Finn pakken basert på avsender, mottaker, lokasjon og kollinummer. Skann kolliet til linjen blir grønn og trykk `Fortsett`.

![TODO: Skjermbilde – skann kolli før ID-kontroll](/img/pda/utlevering-id-kontroll-02-skann-kolli.png)

### Steg 3: Utfør ID-kontroll

Sjekk at ID stemmer med mottakeren på skjermen. Tast inn de tre siste sifrene i fødselsnummeret, eller bruk knappen for ID uten norsk fødselsnummer. Hvis et annet navn vises i PDA-en, åpne nedtrekksmenyen for å se om fullmakt finnes.

![TODO: Skjermbilde – utfør ID-kontroll](/img/pda/utlevering-id-kontroll-03-id-kontroll.png)

### Steg 4: Ta imot signatur

Be kunden signere på skjermen når ID-kontrollen er godkjent.

![TODO: Skjermbilde – signatur etter ID-kontroll](/img/pda/utlevering-id-kontroll-04-signatur.png)

### Steg 5: Trykk OK og lever ut pakken

Trykk `OK`. Lever først ut pakken når sluttbildet bekrefter at pakken kan utleveres.

![TODO: Skjermbilde – levert etter ID-kontroll](/img/pda/utlevering-id-kontroll-05-levert.png)

## Kontrollpunkt

- Kontroller nøye at ID stemmer overens med pakken mottaker.
- Hvis en annen person henter pakken, skal vedkommende vise både egen ID og mottakers ID når PDA krever dette.
- Utlever aldri før `OK` er trykket og leveringsbildet vises.

## Vanlige feil

- Navnet på skjermen stemmer ikke med personen foran deg: sjekk om riktig fullmakt ligger i nedtrekksmenyen.
- Norsk fødselsnummer mangler: bruk alternativet for ID uten norsk fødselsnummer.
- PDA henger i kontrollbildet: se [PDA-problemer](./pda-problemer).
